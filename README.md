## 🎯 What is CSS Battles?

[CSS Battles](https://cssbattle.dev) is a competitive programming platform where players create pixel-perfect layouts and designs using only CSS. The goal is to match target designs as closely as possible with minimal code and CSS properties.

### Key Features:

- **Code Golf**: Write the shortest CSS possible to match the target
- **No JavaScript**: Pure CSS solutions only
- **Main Battles**: Permanent challenges in the platform
- **Daily Targets**: New daily challenges released regularly

## 🚀 How to Use

1. **View a Solution**: Open any `.html` file in your browser to see the CSS battle solution in action.

2. **Study the Code**: Right-click → Inspect (or press F12) to view the source code and CSS used.

3. **Compare with Original**: Visit [CSS Battles](https://cssbattle.dev) to compare your solutions and check rankings.

4. **Create Your Own**: Use these as reference and attempt the challenges yourself on cssbattle.dev!

## 💡 Tips for CSS Battles

- **Minimize CSS**: Use shorthand properties and abbreviations
- **Use CSS Variables**: `--var` can help reduce repetition
- **Master CSS Features**: Box-shadow, gradients, transforms, and clip-path are powerful
- **Practice Daily**: The daily targets are great for building skills
